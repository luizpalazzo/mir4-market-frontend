# MIR4 Market Frontend

Frontend moderno e elegante para exibir NFTs do mercado MIR4.

## Tecnologias

- React 18
- Vite
- Tailwind CSS

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
